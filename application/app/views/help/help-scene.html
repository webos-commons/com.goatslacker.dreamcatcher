Backing up your Data
Currently there are two options provided: Text and JSON. Both are emailed to an address you provide, from there you can safely store your data wherever you wish.
Text
Plaintext format of your dreams. This is useful if you'd like a log of all your dreams to read later elsewhere
JSON
You can use this format to later import your dreams into Dreamcatcher in case you lose all your data. See the section on 'Recover Data'
Data Management
You're able to delete, backup or import data via the Preferences screen. Data is stored in JSON format on the device.
Recover Data
To successfully recover your data you'll need a .json file with your dreams loaded onto your device.
Backing up
  1. Select Email JSON from the main screen
  2. Email it to yourself
  3. On a computer, copy all of the contents in the email and save them to a file with the extension .json
  4. Plug your device into the computer and select USB Drive mode
  5. Drag and drop the .json file onto your device
Importing Data
  1. Go to your preferences and selected "Load Backup Data"
  2. Find the JSON file and tap on it
  3. ...the app will do some work and you'll hopefully have all your dreams
Password
Please try to remember your password, there is no forgot your password feature.
If you forget your password, tap on the Application menu and selected 'Forgot Password' the password will be reset to 'dream'
